# Overview
---
Billing desk Module is concerned with all the payments in which are needed to be done once the service is to be given to the patient or it has already being given.

Also in this module is where some exemption for some patient can be done so that they can get service depending on the criteria that leads to the exemption

It consist of Two Main categories which are
 * <strong>CASHIER</strong>
 * <strong>SOCIAL WELFARE</strong>

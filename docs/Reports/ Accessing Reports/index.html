Steps To Access Reports

  1. After Successful Login to iCare as a Facility Aministrator navigate to Report Module and Click it as it will redirect to reports page

  1. All the Report are grouped into Report Categoris under Data Section To access a desired Report you have to specify first the group(categpry) to which the report belong

  2. Select the Report Category

eg: When Register Report Category is Chosen

Then it will display all the reports found under Register

  1. Select one of the reports eg . Patient attended

  1. Select period to which you what the produced data to cover ie. you want list of patient attended from when to when. This is done by specifying Start Date & Time and End Date & Time

  2. After that then Click the Get Report Button

    Then a Report will be loaded based on the specified Period
